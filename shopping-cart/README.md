# 🛒 Smart Shopping Cart (PyQt5)

A simple desktop shopping cart application built with Python and PyQt5.

## ✨Features
- Add / remove items
- Cart summary
- Special discount items
- Coupon system
- GUI interface

## 💻Tech Stack
- Python
- PyQt5

## 🚀How to Run

```bash
pip install -r requirements.txt
python app.py
```

## 📝Learning Goals
- OOP in Python
- GUI development
- Basic state management

## 📜Project Attribution
- This project is based on a GUI template provided in a coursework setting.
- All core functionality, including shopping cart logic, discount system, and coupon handling, was designed and implemented independently.
